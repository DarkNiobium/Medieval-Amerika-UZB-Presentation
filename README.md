# Medieval Amerika Presentation

This is a Scrollytelling historical presentation website built with React, Vite, Tailwind CSS, and Framer Motion.

## Setup

1.  **Install dependencies** (if you haven't already):
    ```bash
    npm install
    ```

2.  **Add Video Files**:
    Place your video files in the `public` folder with the following names:
    - `video1.mp4`
    - `video2.mp4`
    - `video3.mp4`
    - `video4.mp4`
    - `video5.mp4`
    - `video6.mp4`

3.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

## Features

- **Scrollytelling Experience**: Navigate through history with smooth transitions.
- **Video Backgrounds**: Full-screen video backgrounds for each slide.
- **Interactive Navigation**: Use the arrow keys, mouse wheel, or on-screen buttons to navigate.
- **Glassmorphism UI**: Modern and elegant interface.

## Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
